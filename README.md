# Projects

## 1️⃣ Algerian Forest Fires Prediction (Classification)

**Objective**

Predict whether a forest fire occurred based on meteorological and fire index features.

**Dataset**

Algerian Forest Fires Dataset containing weather observations and fire indices.

**Workflow**

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization
* Feature selection and preparation
* Model training and evaluation

**Machine Learning Models Used**

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* K-Nearest Neighbors (KNN)

**Key Insight**

Random Forest performed the best as it captured the **non-linear relationships between weather variables and fire occurrence**.

## 2️⃣ CO2 Emissions Prediction (Regression)

**Objective**

Predict vehicle **CO2 emissions (g/km)** based on engine specifications and fuel consumption.

**Dataset**

Vehicle emissions dataset containing attributes such as engine size, cylinders, and fuel consumption.

**Workflow**

* Data exploration and cleaning
* Visualization of relationships between features
* Feature engineering
* Model training and comparison

**Machine Learning Models Used**

* Linear Regression
* Ridge Regression
* Decision Tree Regressor
* Random Forest Regressor

**Key Insight**

Fuel Consumption (Combined) is the **strongest predictor of CO2 emissions**, which aligns with real-world expectations.
